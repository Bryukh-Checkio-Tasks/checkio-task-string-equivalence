In unicode, a glyph could be represented in different binary form, and there is only one best way to check the equivalence.

Due to historical reasons, some glyphs were brought into unicode multiple times with same or different forms, therefore the problem of equivalence emerged.

 Canonical equivalence cases defined in UAX #15

Canonical equivalent glyphs must be mutual exchangable under any circumstances, so cyrillic "A" is not equivalent to latin "A".

Input: Two arguments. Both are unicode string

Output: Bool.

Example: 

checkio(u"\u212B", u"\u00C5") == True
checkio(u"\u212B", u"A") == False

How it is used: If you are making some social platform and want to have unique username but not restricted in roman characters, this is this is the least you should know about.
